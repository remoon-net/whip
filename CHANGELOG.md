# Changelog

## [0.5.1] - 2025-01-30

- 使用 fork 后的 websocket 解锁更多使用场景

## [0.5.0] - 2025-01-30

- 抛弃无意义的 peer 选项

## [0.4.0] - 2025-01-30

- 使用 websocket close code 避免在浏览器端抓不到 resp 错误信息的问题

## [0.3.3] - 2025-01-29

- 支持 Subprotocols

## [0.3.2] - 2025-01-27

- 添加 websocket 支持测试

## [0.3.1] - 2025-01-20

- 导出 ServerRejected 错误以便复用错误类型

## [0.3.0] - 2025-01-19

- 添加 ServerRejected 错误, 出现此错误时意味着不要重试

## [0.2.1] - 2024-04-01

- 返回 Subprotocols, 浏览器需要至少命中一个 Subprotocols 才可保持链接
- 添加浏览器测试

## [0.2.0] - 2024-03-22

- 修改 peer id 获取逻辑, 现在在浏览器中应该也能访问

## [0.1.0] - 2024-03-22
